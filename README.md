# RentalManager

RentalManager is a mobile application designed for landlords to efficiently manage rental properties. The app provides functionalities to handle tenant assignments, property occupancy, and payment tracking, all using Jetpack Compose and Firebase Realtime Database.

## Features
- Manage rental properties and tenant details.
- Assign tenants to properties and track occupancy status.
- Record and view rental payments.
- Firebase Realtime Database integration for storing data.
